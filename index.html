Si el Juego que pida pesa mas que 2GB se le mandara en un archivo.zip, que dentro abrá un archivo.exe y varios archivos.bin, extraiga los archivos del zip en una carpeta y ejecute el archivo.exe

Si el juego requiere de Steam para ejecutarlo, se le mandara en un archivo.zip, que dentro abrá un archivo.lua y un archivo.manifest (La mayoria de las veces), para instalar el Juego necesita:
-Steam
-Steamtools
una vez que tenga las aplicaciones necesarias para instalar el juego, extraiga los archivos del zip en una carpeta y arrastre los archivos que estan dentro de la carpeta a la ventana de Steamtools, reinicie Steam y Listo

Los Juegos se pueden pedir en formato Lua y Manifest o un setup para instalar el Juego
